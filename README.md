# 90mins Pitch Pulse Privacy Policy

Production-ready one-page React privacy policy for the 90mins Pitch Pulse football channel.

## Scripts

```bash
npm install
npm run dev
npm run build
```

The page uses the channel logo and banner from `public/assets/`.

## Vercel deployment

This project is ready to deploy from GitHub to Vercel.

- Framework preset: Vite
- Install command: `npm install`
- Build command: `npm run build`
- Output directory: `dist`

The root `vercel.json` file pins those settings and includes an SPA rewrite so direct visits to deployed paths render the React app.
